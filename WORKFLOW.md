# User Workflows & Processes

## Entrepreneur Workflow

### 1. Registration & Verification
1. Visit landing page
2. Click "Register as Entrepreneur"
3. Enter email and password
4. Verify email via link sent to inbox
5. Account activated and redirected to entrepreneur dashboard

### 2. Submit Business Idea (Quick Method)
1. Go to "New Business" page
3. Choose business plan PDF (max 2MB)
4. Submit
5. Status shows as "Pending Review"
6. Admin reviews and approves/rejects
7. Cannot see public listing - admin creates separate one

### 3. Complete Intake Form (Comprehensive Application)
1. Go to "Intake Forms"
2. Start multi-step wizard
3. **Step 1 - Founder Info**: Enter name, email, location
4. **Step 2 - Employment**: Current status, years of experience
5. **Step 3 - Business Concept**: Name, category, description
6. **Step 4 - Market Analysis**: Target market, size, competition
7. **Step 5 - Funding Needs**: Amount needed, use of funds
8. **Step 6 - Commitment**: Agree to terms and conditions
9. Submit form
10. Track status in intake history

### 4. Track Submissions
1. View all submissions in dashboard
2. See status: Pending, Approved, Rejected, Submitted
3. Read admin feedback if rejected
4. View created public listing after approval (linked from admin)

## Admin Workflow

### 1. Setup & Configuration
1. Login with admin credentials
2. Go to "Categories" management
3. Create business categories (e.g., Tech, Finance, Healthcare)
4. Set registration fees per category
5. Platform ready for entrepreneurs

### 2. Review Submissions
1. Go to "Businesses" → "Submissions" tab
2. View entrepreneur submissions
3. Open submission to review:
   - Business name
   - Category
   - Business plan PDF
4. Click "Review" button
5. Read business plan and founder info
6. Approve or reject with feedback
7. If rejected, feedback sent to entrepreneur

### 3. Create Public Listings
1. Go to "Businesses" → "Create Listing" tab
2. Select approved entrepreneur submission (or create from scratch)
3. Fill in public-facing details:
   - Description (marketing pitch)
   - Image (Cloudinary uploaded)
   - Supporting documents
   - Funding needed
   - Equity percentage
4. Submit
5. Listing immediately visible to investors

### 4. Manage Users
1. Go to "Users"
2. View all registered users
3. See user role and status
4. Manage user accounts if needed

### 5. Track Investments
1. Go to "Investments"
2. View all investor commitments
3. See which businesses are raising capital
4. Monitor total investment per business

## Investor Workflow

### 1. Discovery (No Account Required)
1. Visit landing page
2. See featured public businesses
3. Click on business to view details
4. See description, funding goal, equity %
5. Requires account to invest

### 2. Registration
1. Click "Register as Investor"
2. Enter email and password
3. Verify email
4. Account activated

### 3. Browse Opportunities
1. Go to "Browse Businesses"
2. View all public, approved listings
3. Filter by category, funding needed, etc.
4. Click business card for full details
5. See founder info, market analysis, use of funds

### 4. Make Investment
1. Open business listing
2. Review full details
3. Click "Invest Now"
4. Enter investment amount
5. Confirm and submit
6. Investment recorded as "Pending"

### 5. Manage Portfolio
1. Go to "My Investments"
2. View all current investments
3. See business name, amount, status
4. Track investment progress
5. Download investment documentation

## Key Process Differences

| Aspect | Entrepreneur | Admin | Investor |
|--------|--------------|-------|----------|
| **Can View** | Own submissions only | All submissions & listings | Public listings only |
| **Creates** | Submissions (name, category, plan) | Public listings (full details) | Investments |
| **Approves** | N/A | Reviews & approves submissions | N/A |
| **Can See Business Plan** | Yes (their own) | Yes (during review) | No |
| **Public Listings Visible** | Yes (those created by admin) | Yes (ones they create) | Yes (all) |

## Email Notifications

**Entrepreneur receives:**
- Account verification email
- Submission status change (approved/rejected)
- Feedback on rejected submission
- Public listing created notification

**Admin receives:**
- New submission notification
- Investment activity summary

**Investor receives:**
- Account verification email
- Investment confirmation
- Investment status updates
